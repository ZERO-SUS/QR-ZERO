# ⚡ ZERO SUS QR GENERATOR

A sleek, modern, and responsive QR Code Generator built with **HTML, CSS, and JavaScript**. Instantly generate high-quality QR codes for URLs, text, usernames, messages, Wi-Fi details, and more.

Designed with a premium **glassmorphism UI**, animated particles, and a red cyber-inspired aesthetic.

---



## ✨ Features

* ⚡ Instant QR Code generation
* 🌐 Supports URLs, text, usernames, emails, and messages
* 📥 Download generated QR code as PNG
* 🎨 Modern Glassmorphism Design
* 🔥 Animated floating particle background
* ❤️ Red glowing cyber aesthetic
* 📱 Fully responsive layout
* 🚀 Lightweight and fast
* 🖥️ No backend required

---

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (Vanilla)
* QRCode.js

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/ZERO-SUS-QR-GENERATOR.git
```

### Open the project

Simply open **index.html** in your browser.

No installation required.

---

## 🎯 How to Use

1. Enter any text or URL.
2. Click **Generate QR Code**.
3. Your QR code is generated instantly.
4. Click **Download QR** to save it as a PNG image.
